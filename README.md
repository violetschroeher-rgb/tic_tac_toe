# tic_tac_toe
Project description
This is an in-terminal tic-tac-toe game. Players 'X' and 'O' take turns selecting locations on the board until one player wins or there is a draw. The input must be a number between 0 and 2 describing a location on the board. After each turn, the board is updated and the current player switches. 

Instructions to run the program
To play the game, run the file. After running the file the game will start and it will alternate between the players automatically.
